Troubleshooting

  • File names are case-sensitive - Make sure the file names match exactly (e.g., "Brooks-Kelly.jpg" is not the same as "brooks-kelly.jpg")
  • File extensions matter - .jpg, .jpeg, .png are different formats
  • File location - Images must be in the "public" folder, not in "public/images" or any subfolder
  • Hard refresh - Try a hard refresh (Ctrl+F5 or Cmd+Shift+R) to clear cached images
  • Check file size - Very large images may take longer to load
